双指针

双指针，指的是在遍历对象的过程中，不是普通的使用单个指针进行访问，而是使用两个相同方向（快慢指针）或者相反方向（对撞指针)的指针进行扫描，从而达到相应的目的。 换言之，双指针法充分使用了数组有序这一特征，从而在某些情况下能够简化一些运算.

判断子序列-力扣 392 题

给定字符串 s 和 t ，判断 s 是否为 t 的子序列。字符串的一个子序列是原始字符串删除一些（也可以不删除）字符而不改变剩余字符相对位置形成的新字符串。（例如，”ace”是”abcde”的一个子序列，而”aec”不是）.

解题思路

  1. 针对要判断 s 是否是 t 的子序列，则需要判断 s 字符串在 t 字符串中存在或则 s 的字符组成方式需要在 t 中能找到。
  2. 针对这种，则考虑使用双指针的方式，定义指针 i 指向 s，指针 j 指向 t。
  3. 则定义当前字符 cur，针对 s[i] === t[j]的时候 i 指针向右移动，同时 j 指针向右移动，并将此时的 t[j]拼接到字符串 cur 中
  4. 最后将拼接的字符串 cur 和 s 做对比,一致则是子序列，否则不是子序列
function isSubsequence(s: string, t: string): boolean {
  let i = 0,
    j = 0;
  let cur = '';
  while (i < s.length && j < t.length) {
    if (s[i] === t[j]) {
      cur = cur + t[j];
      i++;
    }
    j++;
  }
  return cur === s;
}

训练计划（1）

教练使用整数数组 actions 记录一系列核心肌群训练项目编号。为增强训练趣味性，需要将所有奇数编号训练项目调整至偶数编号训练项目之前。请将调整后的训练项目编号以 数组 形式返回。

思路

考虑定义双指针 i , j 分列数组左右两端，循环执行：

  1. 指针 i 从左向右寻找偶数；
  2. 指针 j 从右向左寻找奇数；
  3. 将 偶数 nums[i] 和 奇数 nums[j] 交换。

算法流程：
初始化： i , j 双指针，分别指向数组 nums 左右两端；
循环交换： 当 i=j 时跳出；

指针 i 遇到奇数则执行 i=i+1 跳过，直到找到偶数；
指针 j 遇到偶数则执行 j=j−1 跳过，直到找到奇数；
交换 nums[i] 和 nums[j] 值；
返回值： 返回已修改的 nums 数组。

function trainingPlan(actions: number[]): number[] {
  if (!actions.length) {
    return [];
  }
  let left = 0,
    right = actions.length - 1;
  while (left < right) {
    // 如果右侧是偶数，则跳过, 在循环内最终有可能left = right指针，所以这里要判断
    while (left < right && actions[right] % 2 === 0) {
      right--;
    }
    // 如果左侧是奇数，跳过
    while (left < right && actions[left] % 2 !== 0) {
      left++;
    }
    [actions[right], actions[left]] = [actions[left], actions[right]];
  }
  return actions;
}

移动零

给定一个数组 nums，编写一个函数将所有 0 移动到数组的末尾，同时保持非零元素的相对顺序。

请注意 ，必须在不复制数组的情况下原地对数组进行操作。

思路

使用双指针，左指针指向当前已经处理好的序列的尾部，右指针指向待处理序列的头部。
右指针不断向右移动，每次右指针指向非零数，则将左右指针对应的数交换，同时左指针右移。
注意到以下性质：

  1. 左指针左边均为非零数；
  2. 右指针左边直到左指针处均为零。

因此每次交换，都是将左指针的零与右指针的非零数交换，且非零数的相对顺序并未改变。

function moveZeroes(nums: number[]): void {
  if (nums.length < 2) {
    return;
  }
  let left = 0,
    right = 0;
  while (right < nums.length) {
    if (nums[right] !== 0) {
      [nums[right], nums[left]] = [nums[left], nums[right]];
      left++;
    }
    right++;
  }
}

代码二

function moveZeroes(nums: number[]): void {
  if (nums.length < 2) {
    return;
  }
  let j = 0;
  for (let i = 0; i < nums.length; i++) {
    if (nums[i] !== 0) {
      [nums[j], nums[i]] = [nums[i], nums[j]];
      j++;
    }
  }
}

盛最多水的容器

给定一个长度为 n 的整数数组 height 。有 n 条垂线，第 i 条线的两个端点是 (i, 0) 和 (i, height[i]) 。
找出其中的两条线，使得它们与 x 轴共同构成的容器可以容纳最多的水。
返回容器可以储存的最大水量。

说明：你不能倾斜容器。

思路

由于盛水最多，那么就需要容器底部最大，即（碗口原理）

初始化： 双指针 i, j 分列水槽左右两端；
循环收窄： 直至双指针相遇时跳出；
更新面积最大值 resresres ；
选定两板高度中的短板，向中间收窄一格；
返回值： 返回面积最大值 resresres

function maxArea(height: number[]): number {
  let left = 0,
    right = height.length - 1,
    ans = 0;
  while (left < right) {
    const width = right - left;
    const realHeight = Math.min(height[left], height[right]);
    ans = Math.max(width * realHeight, ans);
    // 不断往两边收窄，然后比较最大值
    if (height[left] < height[right]) {
      left++;
    } else {
      right--;
    }
  }
  return ans;
}

三数之和

给你一个整数数组 nums ，判断是否存在三元组 [nums[i], nums[j], nums[k]] 满足 i != j、i != k 且 j != k ，同时还满足 nums[i] + nums[j] + nums[k] == 0 。请你返回所有和为 0 且不重复的三元组。

注意：答案中不可以包含重复的三元组

思路

首先对数组进行排序，排序后固定一个数 nums[i]，再使用左右指针指向 nums[i]后面的两端，数字分别为 nums[L] 和 nums[R]，计算三个数的和 sum 判断是否满足为 0，满足则添加进结果集
如果 sum 大于 0, 则将右指针减一
如果 sum 小于 0, 将左指针+1;

由于该处理，会把重复的加进来，所以在 sum 为 0 时直接使用 Set 去重即可；

function threeSum(nums: number[]): number[][] {
    if (nums.length < 3){
        return [];
    }
    // 创建set，在加入的时候去重处理
    const numsSet = new Set<string>();
    nums.sort((a, b) => a - b);
    for (let i = 0; i < nums.length; i ++){
        // 以i为基点，定义左右指针
        let left = i + 1, right = nums.length - 1;
        while(left < right){
            const sum = nums[i] + nums[left] + nums[right];
            if (sum < 0){
                left ++;
                continue;
            }
            if (sum > 0){
                right --;
                continue;
            }
            // sum为0，加到set中
            numsSet.add(`${nums[i]}, ${nums[left]}, ${nums[right]}`)
            left ++;
            right --;
        }
    }
    // 这里对加入到set中的数据再还原会数组
    return [...numsSet].map(item => item.split(',')) as unknown as number[][];
};

接雨水

给定 n 个非负整数表示每个宽度为 1 的柱子的高度图，计算按此排列的柱子，下雨之后能接多少雨水。

上面是由数组 [0,1,0,2,1,0,1,3,2,1,2,1] 表示的高度图，在这种情况下，可以接 6 个单位的雨水（蓝色部分表示雨水）

思路

维护两个指针 left 和 right，以及两个变量 leftMax 和 rightMax 初始时 left=0,right=n−1,leftMax=0,rightMax=0,。指针 left 只会向右移动，指针 right 只会向左移动，在移动指针的过程中维护两个变量 leftMax 和 rightMax 的值。

当两个指针没有相遇时，进行如下操作：

使用 height[left] 和 height[right] 的值更新 leftMax 和 rightMax\的值；

如果 height[left] < height[right]，则必有 leftMax < rightMax，下标 left 处能接的雨水量等于 leftMax−height[left]，将下标 left 处能接的雨水量加到能接的雨水总量，然后将 left 加 1（即向右移动一位）；

如果 height[left] ≥ height[right]，则必有 leftMax ≥ rightMax，下标 right 处能接的雨水量等于 rightMax−height[right]，将下标 right 处能接的雨水量加到能接的雨水总量，然后将 right 减 1（即向左移动一位）

function trap(height: number[]): number {
  if (height.length === 0) {
    return 0;
  }
  let max: number = 0;
  const length = height.length;
  let left = 0,
    right = length - 1;
  let l_max = height[0],
    r_max = height[length - 1];
  while (left < right) {
    l_max = Math.max(l_max, height[left]);
    r_max = Math.max(r_max, height[right]);
    if (l_max < r_max) {
      max += l_max - height[left];
      left++;
    } else {
      max += r_max - height[right];
      right--;
    }
  }
  return max;
}

无重复字符的最长子串

给定一个字符串 s ，请你找出其中不含有重复字符的 最长子串 的长度。

如下:
输入: s = “pwwkew”
输出: 3
解释: 因为无重复字符的最长子串是 “wke”，所以其长度为 3。

请注意，你的答案必须是 子串 的长度，”pwke” 是一个子序列，不是子串。

思路

一、双指针

定义指针 i 和 j，j 的初始值是 i + 1，以 i 为基准，让 j 往后循环，并且将 j 的字符拼到临时字符上
之后记录字符长度和结果长度，两者取最大值
返回结果即可

function lengthOfLongestSubstring(s: string): number {
  let ans = 0;
  if (s.length < 2) {
    return s.length;
  }
  for (let i = 0; i < s.length; i++) {
    let resStr = s[i],
      j = i + 1;
    while (j < s.length) {
      if (resStr.includes(s[j])) {
        break;
      }
      resStr = `${resStr}${s[j]}`;
      j++;
    }
    ans = Math.max(ans, resStr.length);
  }
  return ans;
}

二、滑动窗口

滑动窗口其实就是一个队列,比如例题中的 abcabcbb，进入这个队列（窗口）为 abc 满足题目要求，当再进入 a，队列变成了 abca，这时候不满足要求。所以，我们要移动这个队列！

只要把队列的左边的元素移出就行了，直到满足题目要求！

function lengthOfLongestSubstring(s: string): number {
  if (!s) {
    return 0;
  }
  let strArr: string[] = [];
  let max: number = 0;
  for (let i = 0; i < s.length; i++) {
    const cur = s.charAt(i);
    const index = strArr.findIndex((x) => x === cur);
    if (index !== -1) {
      strArr.splice(0, index + 1);
    }
    strArr.push(cur);
    max = Math.max(max, strArr.length);
  }
  return max;
}

找到字符串中所有字符的异位词

思路

方法一: 窗口暴力法

  1. 先将 p 字符排序
  2. 循环字符 s,往窗口中加入字符，当窗口长度与 p 相同，排序窗口并比较，如果一直，则将索引加入结果
  3. 返回结果
function findAnagrams(s: string, p: string): number[] {
  if (!s || s.length < p.length) {
    return [];
  }
  const ans = [];
  // 排序p字符
  const sortStr = p.split('').sort().join('');
  for (let i = 0; i < s.length; i++) {
    const cur = s[i];
    // 如果当前字符不在p中，直接往下
    if (!sortStr.includes(cur)) {
      continue;
    }
    const temp = [];
    let curIndex = i;
    // 往窗口加入字符
    while (
      temp.length !== p.length &&
      s[curIndex] &&
      sortStr.includes(s[curIndex])
    ) {
      temp.push(s[curIndex]);
      curIndex++;
    }
    const target = temp.sort().join('');
    // 相等则加入索引
    if (target === sortStr) {
      ans.push(i);
    }
  }
  return ans;
}

但是会发现，其实字符被多次比较了，所以当字符很大时，会超时

方法二: 滑动窗口 + 哈希表

可以先统计出字符串 p 中每个字符的词频数，再把 p 的长度作为滑动窗口的宽度，把这个窗口在字符串 s 中移动，统计窗口中的词频数，如果在移动的过程中，窗口中的词频数与 p 的词频数相等，那么，这个窗口就是 p 的一个异位词。

滑动窗口的最大值

给你一个整数数组 nums，有一个大小为 k 的滑动窗口从数组的最左侧移动到数组的最右侧。你只可以看到在滑动窗口内的 k 个数字。滑动窗口每次只向右移动一位。
返回 滑动窗口中的最大值 。

function maxSlidingWindow(nums: number[], k: number): number[] {
  if (!nums.length || k < 1 || nums.length < k) {
    return [];
  }
  const ans = [];
  const queue = []; // 双端队列，内部存储索引,并且要保持队列中数据是单调递减的
  let index = 0;
  for (let i = 0; i < nums.length; i++) {
    // i 表示窗口的右边界

    // 循环表示队列中最后一个数的下标是否大于nums[i]的值，如果不大于，则将队列中该值删掉
    while (queue.length && nums[queue[queue.length - 1]] <= nums[i]) {
      queue.pop();
    }
    queue.push(i);
    // i - k的下标过期，即窗口左值往右移动
    if (queue[0] === i - k) {
      queue.shift();
    }
    if (i >= k - 1) {
      // 窗口构建完成
      ans[index] = nums[queue[0]];
      index++;
    }
  }
  return ans;
}

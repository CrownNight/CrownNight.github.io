合并两个有序链表

将两个升序链表合并为一个新的 升序 链表并返回。新链表是通过拼接给定的两个链表的所有节点组成的。

  • 思路
    1. 针对要合并两个链表，并且还要按顺序合并，其实也不难
    2. 简单来说，就定义一个结果链表 ans，然后循环这两个链表
    3. 循环过程中，比较两个链表当前节点的值，小的就拼接到 ans 中，移动小的链表，继续循环，直到某一个链表为空
    4. 循环完成后，由于两个链表长度可能不相等，所以需要在最后将剩余的链表拼上。
    5. 还有一个注意点，由于是创建的一个新链表，而这个新链表是从 0 开始的，我们是不需要这个 0 节点的，所以最后返回 ans.next。
function mergeTwoLists(
  list1: ListNode | null,
  list2: ListNode | null
): ListNode | null {
  const ans = new ListNode();
  let cur = ans;
  while (list1 && list2) {
    if (list1.val < list2.val) {
      cur.next = list1;
      list1 = list1.next;
    } else {
      cur.next = list2;
      list2 = list2.next;
    }
    cur = cur.next;
  }
  // 处理完毕之后，至多还有一个没有处理
  cur.next = list1 ? list1 : list2;
  // 返回next，因为ans的第一个节点为0
  return ans.next;
}

反转链表

给你单链表的头节点 head ，请你反转链表，并返回反转后的链表。

;

  • 思路
    1. 假设链表为 1 -> 2 -> 3 -> null，我们想要把它改 3 -> 2 -> 1 -> null.
    2. 在遍历链表时，将当前节点的 next 指针改为指向前一个节点。
    3. 由于节点没有引用其前一个节点，因此必须事先存储其前一个节点。
    4. 在更改引用之前，还需要存储后一个节点。
    5. 最后返回新的头引用。
    6. 其实反转链表这是一个统一的格式，记住更好
/**
 * Definition for singly-linked list.
 * class ListNode {
 *     val: number
 *     next: ListNode | null
 *     constructor(val?: number, next?: ListNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.next = (next===undefined ? null : next)
 *     }
 * }
 */

function reverseList(head: ListNode | null): ListNode | null {
  let pre = null;
  let cur = head;
  while (cur) {
    const nextNode = cur.next;
    cur.next = pre;
    pre = cur;
    cur = nextNode;
  }
  return pre;
}

链表的第一个公共节点

解题思路

使用两个指针 N1,N2，一个从链表 1 的头节点开始遍历，我们记为 N1，一个从链表 2 的头节点开始遍历，我们记为 N2。

让 N1 和 N2 一起遍历，当 N1 先走完链表 1 的尽头（为 null）的时候，则从链表 2 的头节点继续遍历，同样，如果 N2 先走完了链表 2 的尽头，则从链表 1 的头节点继续遍历，也就是说，N1 和 N2 都会遍历链表 1 和链表 2。

因为两个指针，同样的速度，走完同样长度（链表 1+链表 2），不管两条链表有无相同节点，都能够到达同时到达终点。

（N1 最后肯定能到达链表 2 的终点，N2 肯定能到达链表 1 的终点）。

所以，如何得到公共节点：

有公共节点的时候，N1 和 N2 必会相遇，因为长度一样嘛，速度也一定，必会走到相同的地方的，所以当两者相等的时候，则会第一个公共的节点
无公共节点的时候，此时 N1 和 N2 则都会走到终点，那么他们此时都是 null，所以也算是相等了。

代码

/*class ListNode {
 *     val: number
 *     next: ListNode | null
 *     constructor(val?: number, next?: ListNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.next = (next===undefined ? null : next)
 *     }
 * }
 */
/**
 * @param pHead1 ListNode类
 * @param pHead2 ListNode类
 * @return ListNode类
 */
export function FindFirstCommonNode(
  pHead1: ListNode,
  pHead2: ListNode
): ListNode {
  let l1 = pHead1,
    l2 = pHead2;
  while (l1 !== l2) {
    l1 = l1 === null ? pHead2 : l1.next;
    l2 = l2 === null ? pHead1 : l2.next;
  }
  return l1;
}

链表中环的入口

题目描述

给一个链表，若其中包含环，请找出该链表的环的入口结点，否则，返回 null。
输入描述：输入分为 2 段，第一段是入环前的链表部分，第二段是链表环的部分，后台将这 2 个会组装成一个有环或者无环单链表
返回值描述：返回链表的环的入口结点即可。而我们后台程序会打印这个节点
示例 1
输入：{1,2},{3,4,5}
返回值：3
说明：返回环形链表入口节点，我们后台会打印该环形链表入口节点，即 3

解题思路

两种方案：

  • 使用字典存储链表走的节点，如果字典中第一次找到重复的值，则该值就是环的入口位置

  • 使用快慢指针处理, 定义快慢指针 slow 和 fast 指针，slow 每走一步，fast 走两步，若是有环，则一定会在环的某个结点处相遇（slow == fast）

代码

方案一(字典记录)

/*class ListNode {
 *     val: number
 *     next: ListNode | null
 *     constructor(val?: number, next?: ListNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.next = (next===undefined ? null : next)
 *     }
 * }
 */
/**
 * @param pHead ListNode类
 * @return ListNode类
 */
export function EntryNodeOfLoop(pHead: ListNode): ListNode {
  if (!pHead) return null;
  const listDic: Record<string, ListNode> = {};
  while (pHead) {
    if (listDic[pHead.val]) {
      return pHead;
    }
    listDic[pHead.val] = pHead;
    pHead = pHead.next;
  }
  return null;
}

方案二（快慢指针）

/*class ListNode {
 *     val: number
 *     next: ListNode | null
 *     constructor(val?: number, next?: ListNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.next = (next===undefined ? null : next)
 *     }
 * }
 */
/**
 * @param pHead ListNode类
 * @return ListNode类
 */
export function EntryNodeOfLoop(pHead: ListNode): ListNode {
  if (!pHead) return null;
  let slow = pHead,
    fast = pHead;
  while (fast !== null && fast.next !== null) {
    fast = fast.next.next;
    slow = slow.next;
    // 第一次相遇的节点
    if (slow === fast) {
      break;
    }
  }
  // 这里如果快指针为null，则表明走到链表底了，那么链表就没有环
  if (fast === null || fast.next === null) {
    return null;
  }
  // 重新指向链表头部
  fast = pHead;
  // 与第一次相遇的节点相同速度出发，相遇的时候就是入口节点
  while (fast !== slow) {
    fast = fast.next;
    slow = slow.next;
  }
  return fast;
}

难度中等 删除链表中的重复节点

题目描述

在一个排序的链表中，存在重复的结点，请删除该链表中重复的结点，重复的结点不保留，返回链表头指针。 例如，链表 1->2->3->3->4->4->5 处理后为 1->2->5

数据范围：链表长度满足 0≤n≤1000 ，链表中的值满足 1≤val≤1000

进阶：空间复杂度 O(n)，时间复杂度 O(n)

例如输入{1,2,3,3,4,4,5}时，对应的输出为{1,2,5}，对应的输入输出链表如下图所示

思路

  1. 使用 hash 表记录出现过的数据，如果有重复的,将这一项记录为 true,
  2. 然后创建一个新的链表，重新按 hash 表加入即可

代码

/*class ListNode {
 *     val: number
 *     next: ListNode | null
 *     constructor(val?: number, next?: ListNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.next = (next===undefined ? null : next)
 *     }
 * }
 */

/**
 * @param pHead ListNode类
 * @return ListNode类
 */
export function deleteDuplication(pHead: ListNode): ListNode {
  // 创建字典
  const repeatDic: Record<string, boolean> = {};
  let cur = pHead;
  while (cur) {
    // 如果在字典里，则将该项置为true
    if (Reflect.has(repeatDic, cur.val)) {
      repeatDic[cur.val] = true;
    } else {
      repeatDic[cur.val] = false;
    }
    cur = cur.next;
  }
  let ansNode = new ListNode();
  cur = pHead;
  let ans = ansNode;
  while (cur) {
    if (!repeatDic[cur.val]) {
      ans.next = cur;
      ans = ans.next;
    } else {
      // 这里如果不在hash表里面需要将尾节点置空，否则如果最后的数据是重复，那么无法去掉
      ans.next = null;
    }
    cur = cur.next;
  }
  return ansNode.next;
}

两数相加

题目描述

给你两个 非空 的链表，表示两个非负的整数。它们每位数字都是按照 逆序 的方式存储的，并且每个节点只能存储 一位 数字。
请你将两个数相加，并以相同形式返回一个表示和的链表。
你可以假设除了数字 0 之外，这两个数都不会以 0 开头。

思路

这里链表是逆序的，注意这个。 其次由于数字相加需要考虑进位的情况，当这两个考虑之后，就迎刃而解了

function addTwoNumbers(
  l1: ListNode | null,
  l2: ListNode | null
): ListNode | null {
  // 定义结果链表
  const ans = new ListNode();
  // 这里其实是进位，懒得取名了
  let sum = 0;
  let temp = ans;
  // 这里会有情况，当两个链表都为空了，但是进位还有，所以这里还需判断进位是否为空
  while (l1 || l2 || sum) {
    let sum1 = l1?.val || 0;
    let sum2 = l2?.val || 0;
    // 当前两个链表以及进位相加
    const curSum = sum1 + sum2 + sum;
    // 处理是否还有进位，预存
    sum = Math.floor(curSum / 10);
    temp.next = new ListNode(curSum % 10);
    temp = temp.next;
    if (l1) {
      l1 = l1.next;
    }
    if (l2) {
      l2 = l2.next;
    }
  }
  return ans.next;
}

删除链表的第 N 个节点

题目描述

给你一个链表，删除链表的倒数第 n 个结点，并且返回链表的头结点。

思路

注意，这里是删除第 N 个节点，和删除第 K 个节点的题不一样的，这里只删除指定的一个节点
所以这里，就是先找链表长度，如果链表长度小于第 N 个，则直接返回空，因为不存在

function removeNthFromEnd(head: ListNode | null, n: number): ListNode | null {
  if (!head || n === 0) {
    return head;
  }
  let length = 0;
  let cur = head;
  while (cur) {
    length++;
    cur = cur.next;
  }
  // 链表长度小于n
  if (length < n) {
    return null;
  }
  cur = head;
  const diff = length - n;
  // 差值等于0，表示删除第一个节点
  if (diff === 0) {
    return head.next;
  }
  for (let i = 0; i < diff; i++) {
    if (i === diff - 1) {
      cur.next = cur.next.next;
    }
    cur = cur.next;
  }
  return head;
}

旋转链表

题目描述

给你一个链表的头节点 head ，旋转链表，将链表每个节点向右移动 k 个位置。

思路

旋转链表，就是将链表长度 length 求出，然后把链表连成环，再旋转这个环的链表，最后在移动 length - k % length -1 个节点，再将链表在此处断开即可

为什么是移动 length - k % length -1 个节点
首先 k 可能会很大，所以需要取余， 其次 如果移动的是 length - k % length 个节点，那么就多移动了一个，无法在该节点断开

function rotateRight(head: ListNode | null, k: number): ListNode | null {
  // 处理边界条件
  if (!head || k === 0 || !head.next) {
    return head;
  }
  let length = 0;
  let cur = head;
  // 尾节点，这里需要记录一下尾节点，后续才能将链表连成环
  let endNode = null;
  while (cur) {
    if (cur.next === null) {
      endNode = cur;
    }
    cur = cur.next;
    length++;
  }
  let moveCount = k % length;
  // 如果长度与k整除 则直接返回当前链表即可
  if (moveCount === 0) {
    return head;
  }
  // 将链表连成环
  endNode.next = head;
  // 开始旋转链表，转的次数是链表长度 - 次数 -1
  for (let i = 0; i < length - moveCount - 1; i++) {
    head = head.next;
  }
  // 旋转之后，记录下一个节点
  let headNext = head.next;
  // 将链表下一个节点置为null, 断开链表的环
  head.next = null;
  return headNext;
}

分隔链表

题目描述

给你一个链表的头节点 head 和一个特定值 x ，请你对链表进行分隔，使得所有 小于 x 的节点都出现在 大于或等于 x 的节点之前。

你应当 保留 两个分区中每个节点的初始相对位置。

思路

既然要分隔链表，并且注意题目信息，所有小于 x 的值在大于等于 x 的左边，所以我们可以直接创建两个链表，将小于该 x 的放一个链表，大于等于的放另一个链表
最后再将两个链表连接即可

function partition(head: ListNode | null, x: number): ListNode | null {
  if (!head) {
    return null;
  }
  let cur = head;
  // 创建结果链表，即小于x的头链表
  let ans = new ListNode();
  // 创建尾链表，即大于等于x的
  let endAns = new ListNode();
  let tempAns = ans,
    endTempAns = endAns;
  // 循环
  while (cur) {
    // 小于x的创建新的链表，并且将尾链表下一个节点置为空，防止尾链表后面有小于x的值
    if (cur.val < x) {
      tempAns.next = cur;
      tempAns = tempAns.next;
      endTempAns.next = null;
    } else {
      // 同理，大于等于的创建新链表，并且将小于x的链表尾节点指向null
      tempAns.next = null;
      endTempAns.next = cur;
      endTempAns = endTempAns.next;
    }
    cur = cur.next;
  }
  // 重新指向首链表，找到最后一个节点
  tempAns = ans;
  while (tempAns.next) {
    tempAns = tempAns.next;
  }
  // 最后一个节点指向尾链表
  tempAns.next = endAns.next;
  return ans.next;
}

寄偶链表

给定单链表的头节点 head ，将所有索引为奇数的节点和索引为偶数的节点分别组合在一起，然后返回重新排序的列表。

第一个节点的索引被认为是 奇数 ， 第二个节点的索引为 偶数 ，以此类推。

请注意，偶数组和奇数组内部的相对顺序应该与输入时保持一致。

你必须在 O(1) 的额外空间复杂度和 O(n) 的时间复杂度下解决这个问题。

思路

因为是要在 O(1)的空间复杂度上解决，则不能使用暴力的创建一个新链表，然后根据奇偶性做增加了；
所以这里考虑使用双指针方式解决

  1. 创建奇数指针 odd 并指向 head
  2. 创建偶数指针 even 并执行 head.next
  3. 循环偶数指针，并且偶数指针跟随移动，在循环中将奇数全部放入奇数指针中，同理偶数指针中就只有偶数
  4. 最后将奇数指针的 next 指向偶数指针即可，返回 head;
function oddEvenList(head: ListNode | null): ListNode | null {
  if (!head) {
    return null;
  }
  // 偶数指针
  let even = head.next;
  // 奇数指针，并且再创建一个临时偶数指针执行原有的偶数指针
  let odd = head,
    evenNode = even;
  while (evenNode && evenNode.next) {
    // 计数指针指向偶数的下一个
    odd.next = evenNode.next;
    odd = odd.next;
    // 偶数指针下一个指向计数指针下一个，即偶数当前是偶数，但是下一个是奇数，所以相当于将偶数的下一个节点删除
    evenNode.next = odd.next;
    evenNode = evenNode.next;
  }
  // 最后奇数指针的next执行偶数指针
  odd.next = even;
  return head;
}
